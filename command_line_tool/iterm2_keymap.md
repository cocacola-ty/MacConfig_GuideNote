# iterm2快捷键

* 新建标签

    ```
    command + t
    ```

* 关闭标签

    ```
    command + w
    ```

* 上下分屏

    ```
    command + shift + d
    ```

* 左右分屏

    ```
    command + d 
    ```

* 焦点移动其他分屏

    ```
    command + option + left/right/up/down
    ```

* 查看剪贴板历史 

    ```
    command + shift + h
    ```

* 搜索输入过的命令

    ```
    ctrl + r
    ```

* 将光标移动到行首

    ```
    ctrl + a
    ```
* 将光标移动到行尾

    ```
    ctrl + e
    ```
* 删除光标前的一个单词

    ```
    ctrl + w
    ```
* 从光标位置删除到行尾

    ```
    ctrl + k
    ```

* 删除整行

    ```
    ctrl + u
    ```
