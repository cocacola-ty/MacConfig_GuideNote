
`du`命令，即disk usage，查看硬盘使用情况命令

显示目录或文件占用的磁盘空间

#### 参数 

* `-s`： 仅显示当前目录的总计大小
* `-h`： 以`K`、`M`、`G`为单位，提供可读性
* `-b`：以`byte`为单位

#### 实例

* 显示指定文件的大小

    ```
        du a.txt
    ```
    ```
    >>> 424 a.txt
    ```

* 显示当前目录的总计大小, 并以`K`、`M`、`G`为单位显示

    ```
     du -sh
    ```
    ```
    >>> 31M .
    ```

* 对当前目录的所有文件 按大小排序

    ```
    du -sh * | sort -nr
    ```
    
    ```
    904K	company
    824K	每日分享a 
    332K	1059484C-AB4D-4D15-9219-2C599DB6E2EC.JPG
    ```
